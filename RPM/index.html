OBJETIVO DE RPM


Para RPM la intension es generar una interacción segura, eficiente y confiable entre compradores y vendedores que facilite la comercialización
y adquisición de moto partes en la ciudad de Bogotá y así promover el crecimiento sostenible del negocio. A través del desarrollo de un sistema de
información web con un meticuloso proceso de creación de perfiles verificados, además de realizar una herramienta de búsqueda
y comunicación que complemente la interacción entre comprador y vendedor tanto en la búsqueda de autopartes como en adquirir servicios mecánicos.

OBJETIVOS GENERALES DEL PROYECTO

  1. Optimizar una interfaz para que el usuario navegue en la página web donde pueda buscar,
    comparar y comprar moto partes de manera rápida y fácil.
  2. Implementar una forma de intercambio seguro de moto partes entre
    vendedores y compradores mediante un proceso de creación de cuentas verificadas para ambos tipos de usuarios.
  3. Implementar medidas de seguridad en pro de proteger la integridad e información del cliente donde
    se garantice la seguridad en la información.
  4. Implementar un sistema de reseñas y/o calificaciones en vendedores
    para que así se fomente la confianza entre vendedores y compradores.
  5. Promocionar los negocios locales que se destaquen por su buen
    desempeño en ventas a través de la aplicación.

TECNOLOGIAS

Para este proyecto se plantea usar la opcion de APIS para formular e implementar la opcion de mapa de busqueda de locaciones cercanas a su ubicacion inicial,
con ayuda de python que hasta el momento es el lenguaje que se estara utilizando para el desarrollo del mismo, se desarrollara el programa con el fin de dar
solucion a los conflictos de hoy en dia y que las personas que utilicen la pagina deduzcan que somo la mejor opcion.
